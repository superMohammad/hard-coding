# Python Practice Pack — 200 Problems

This folder contains **200 practice problems** based on these topics from your course screenshot:

- Python Control Flow
- Inbuilt Data Structures
- Functions in Python
- Function Practice
- Data Structure Practice
- Importing, Creating Modules, and Packages
- File Handling in Python
- Exception Handling in Python

## Levels

- **Level 1 — Warm-up**: core basics, direct practice
- **Level 2 — Builder**: mixed logic, edge cases, small design choices
- **Level 3 — Boss Level**: tricky scenarios, messy data, debugging mindset

## How to use this pack

1. Solve in order, or jump topic by topic.
2. For each problem, write:
   - your solution
   - test cases
   - what confused you
   - what you learned
3. If a problem feels weird, good — that usually means it found a gap.
4. Many prompts are intentionally a little funny so your brain stays awake instead of becoming a potato.

## Suggested self-check for every problem

- Did I understand the requirement clearly?
- Did I handle empty input / bad input / odd cases?
- Can I explain why I chose this structure?
- Can I rewrite it more cleanly?
- What would break my solution?
